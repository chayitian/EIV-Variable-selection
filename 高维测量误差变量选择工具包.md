# 高维测量误差变量选择工具包

## 项目概述

本仓库针对**带测量误差的高维线性回归模型（Error-in-Variables, EIV）**，实现了7种变量选择方法，提供了完整的基准测试流程。包含朴素基准模型、带偏差校正的经典方法、凸优化前沿方法、自适应改进方法，以及本项目创新提出的树模型特征重要性结合方法。所有实现均适配Trae AI IDE，可直接用于代码自动生成、运行与性能对比。

带加性测量误差的核心回归模型定义如下：

$$\begin{cases}
y = X\beta^* + \varepsilon \\
W = X + U
\end{cases}$$

其中：

- $y \in \mathbb{R}^n$：响应变量向量

- $X \in \mathbb{R}^{n \times p}$：不可观测的真实协变量矩阵（高维场景下 $p \gg n$）

- $W \in \mathbb{R}^{n \times p}$：含测量误差的可观测协变量矩阵

- $U \in \mathbb{R}^{n \times p}$：零均值测量误差矩阵，已知协方差矩阵 $\Sigma_{uu} = \mathbb{E}[U^TU/n]$

- $\beta^* \in \mathbb{R}^p$：真实稀疏系数向量（仅 $s \ll n$ 个非零元素）

- $\varepsilon \in \mathbb{R}^n$：独立零均值高斯噪声，方差为 $\sigma^2$

## 算法演进逻辑

本工具包的算法按照以下逻辑演进：

```
基础层（无校正）
    ↓
参考算法层（传统校正）
    ↓
自适应改进层
    ↓
树方法创新层
```

## 核心模型与实现规范

所有模型均采用标准化的输入输出接口设计：

- **输入**：`W`（可观测协变量）、`y`（响应变量）、`alpha`（正则化参数）、`Sigma_uu`（测量误差协方差矩阵）

- **输出**：`coef_`（系数估计向量）、`intercept_`（截距项估计值）

### 1. 朴素Lasso（基础基准模型）

#### 模型原理

不对测量误差做任何校正，直接将标准Lasso方法应用于含噪声的协变量矩阵 $W$。该模型作为基准，用于量化测量误差带来的估计偏差。使用**坐标下降法**实现。

#### 目标函数

$$\hat{\beta}_{\text{naive}} = \arg\min_{\beta} \frac{1}{2n}\|y - W\beta\|_2^2 + \lambda\|\beta\|_1$$

其中：

- $\lambda > 0$：正则化参数

- $\|\beta\|_1$：L1范数，用于实现稀疏变量选择

#### 算法实现步骤

1. 对输入 $W$ 和 $y$ 做中心化处理（减去均值）

2. 初始化系数向量 $\beta$ 为全零

3. **坐标下降法迭代**：
   - 对每个坐标 $j$，计算残差（排除当前特征贡献）
   - 通过软阈值算子更新 $\beta_j$：
     $$\beta_j = S(\rho, \lambda) / z$$
     其中软阈值函数 $S(\rho, \lambda) = \text{sign}(\rho) \cdot \max(|\rho| - \lambda, 0)$

4. 迭代至收敛（$\beta$ 的变化量<1e-4）或达到最大迭代次数（1e3）

5. 恢复截距项

### 2. 修正Lasso（参考算法1 - 显式偏差校正）

#### 模型原理

在Lasso目标函数中引入显式的偏差校正项，利用已知的测量误差协方差矩阵 $\Sigma_{uu}$ 校正损失函数的二次项，消除加性测量误差带来的估计偏差。

#### 目标函数

$$\hat{\beta}_{\text{corrected}} = \arg\min_{\beta} \frac{1}{2n}\|y - W\beta\|_2^2 - \frac{1}{2}\beta^T\Sigma_{uu}\beta + \lambda\|\beta\|_1$$

校正项 $-\frac{1}{2}\beta^T\Sigma_{uu}\beta$ 的核心作用是抵消测量误差导致的格拉姆矩阵膨胀（$\mathbb{E}[W^TW/n] = X^TX/n + \Sigma_{uu}$）。

#### 算法实现步骤

1. **数据标准化**：
   - 对输入 $W$ 和 $y$ 做标准化
   - 同步调整 $\Sigma_{uu}$ 以匹配标准化后的尺度：$\Sigma_{uu}^{\text{scaled}} = \Sigma_{uu} / (\sigma_W \sigma_W^T)$

2. **构建校正格拉姆矩阵**：
   $$\Sigma_{\text{corrected}} = \frac{1}{n}W^TW - \Sigma_{uu}$$

3. **半正定修正**：若最小特征值小于0，添加对角矩阵使其半正定

4. **Cholesky分解与变换**：
   - 对 $\Sigma_{\text{corrected}}$ 做Cholesky分解：$\Sigma_{\text{corrected}} = L L^T$
   - 变换数据：$W_{\text{transformed}} = L^T \sqrt{n}$，$y_{\text{transformed}} = L^{-1} \rho \sqrt{n}$（其中 $\rho = W^Ty/n$）

5. **标准Lasso求解**：在变换后的数据上调用 `sklearn.linear_model.Lasso`

6. **逆变换**：将系数变换回原始尺度

### 3. CoCoLasso（参考算法2 - 凸约束Lasso）

#### 模型原理

由Datta & Zou (2017)针对高维测量误差回归提出，核心是将真实格拉姆矩阵的无偏替代矩阵投影到最近的半正定矩阵空间，解决了传统校正方法的非凸问题。该方法保留了Lasso的凸性优势，支持加性/乘性测量误差、随机缺失数据场景，具备严格的统计理论保证。

#### 核心公式

1. **无偏替代矩阵构建（加性误差场景）**

   $$\widehat{\Sigma} = \frac{1}{n}W^TW - \Sigma_{uu}, \quad \tilde{\rho} = \frac{1}{n}W^Ty$$

   其中 $\widehat{\Sigma}$ 是真实格拉姆矩阵 $\Sigma = X^TX/n$ 的无偏估计，但在高维场景下可能出现负特征值。

2. **最近半正定矩阵投影（ADMM算法）**

   $$\tilde{\Sigma} = (\widehat{\Sigma})_+ = \arg\min_{K \geq 0} \|K - \widehat{\Sigma}\|_F$$

   通过交替方向乘子法（ADMM）求解投影问题：
   - $X$ 更新：$X = (\widehat{\Sigma} + \rho Z - U) / (1 + \rho)$
   - $Z$ 更新：特征值分解后将负特征值置零
   - $U$ 更新：$U = U + \rho(X - Z)$

3. **CoCoLasso目标函数**

   $$\hat{\beta}_{\text{coco}} = \arg\min_{\beta} \frac{1}{2}\beta^T\tilde{\Sigma}\beta - \tilde{\rho}^T\beta + \lambda\|\beta\|_1$$

   对 $\tilde{\Sigma}$ 做Cholesky分解后，可等价转化为标准Lasso形式求解。

#### 算法实现步骤

1. 数据标准化与 $\Sigma_{uu}$ 尺度调整

2. 计算无偏替代矩阵 $\widehat{\Sigma}$ 与 $\tilde{\rho}$

3. **ADMM投影**：求解最近半正定矩阵 $\tilde{\Sigma}$

4. Cholesky分解 $\tilde{\Sigma} = L L^T$

5. 数据变换后调用标准Lasso求解

6. 逆变换回原始尺度

#### 代码文件

`src/CoCoLasso.py`

#### 参考文献

Datta, A., & Zou, H. (2017). CoCoLasso for High-dimensional Error-in-variables Regression. The Annals of Statistics, 45(6), 2400-2426.

### 4. 自适应修正Lasso（自适应改进算法1）

#### 模型原理

将修正Lasso的偏差校正项与自适应Lasso框架结合，在L1惩罚项中对不同系数分配差异化权重：对真实值较大的系数施加更小的惩罚，对零系数施加更大的惩罚，最终实现神谕性质（即变量选择相合性、非零系数估计的渐近正态性）。

#### 核心公式

1. **初始估计**：先用修正Lasso得到初始相合估计 $\hat{\beta}_{\text{init}}$

2. **自适应权重计算**

   $$w_j = |\hat{\beta}_{j, \text{init}}|^{-\gamma}, \quad \gamma > 0$$

   默认取 $\gamma=1$，加入极小值避免除零错误：$w_j = 1/(|\hat{\beta}_{j, \text{init}}| + 1e-6)^\gamma$

3. **自适应修正Lasso目标函数**

   $$\hat{\beta}_{\text{ada\_corrected}} = \arg\min_{\beta} \frac{1}{2n}\|y - W\beta\|_2^2 - \frac{1}{2}\beta^T\Sigma_{uu}\beta + \lambda\sum_{j=1}^p w_j|\beta_j|$$

#### 算法实现步骤

1. **步骤1**：在全数据集上拟合修正Lasso，得到初始估计 $\hat{\beta}_{\text{init}}$（使用较小的 $\lambda$，如 $\lambda/10$）

2. **步骤2**：计算自适应权重 $w_j$

3. **步骤3**：加权优化重参数化：
   - 令 $\beta_j = w_j \cdot \alpha_j$，代入目标函数
   - 对协变量矩阵的每一列做变换：$W_j = W_j / w_j$
   - 对 $\Sigma_{uu}$ 做变换：$\Sigma_{uu}^{\text{weighted}} = \text{diag}(w) \Sigma_{uu} \text{diag}(w)$

4. **步骤4**：在变换后的数据上求解带 $\lambda|\alpha|_1$ 惩罚项的修正Lasso

5. **步骤5**：逆变换得到最终估计 $\hat{\beta}_j = w_j \cdot \hat{\alpha}_j$

#### 代码文件

`src/Adaptive_Corrected_Lasso.py`

李锋, 盖宇杰, 卢一强. (2014). 测量误差模型的自适应LASSO变量选择方法研究. 中国科学: 数学, 44(9), 983-1006.

### 5. 自适应CoCoLasso（自适应改进算法2 - 创新模型）

#### 模型原理

本项目创新提出的方法，将CoCoLasso的凸优化框架与自适应Lasso惩罚项结合。既继承了CoCoLasso对含噪声数据的凸性、鲁棒性优势，又通过自适应权重提升变量选择精度，在高维测量误差回归场景下实现神谕性质，是本项目的核心创新模型之一。

#### 核心公式

1. **初始估计**：先用标准CoCoLasso得到初始相合估计 $\hat{\beta}_{\text{coco}}$

2. **自适应权重计算**

   $$w_j = |\hat{\beta}_{j, \text{coco}}|^{-\gamma}, \quad \gamma > 0$$

   默认取 $\gamma=1$。

3. **自适应CoCoLasso目标函数**

   $$\hat{\beta}_{\text{ada\_coco}} = \arg\min_{\beta} \frac{1}{2}\beta^T\tilde{\Sigma}\beta - \tilde{\rho}^T\beta + \lambda\sum_{j=1}^p w_j|\beta_j|$$

   其中 $\tilde{\Sigma}$、$\tilde{\rho}$ 与标准CoCoLasso一致，为半正定投影后的格拉姆矩阵与响应变量替代向量。

#### 算法实现步骤

1. **步骤1**：拟合标准CoCoLasso，得到初始系数估计 $\hat{\beta}_{\text{coco}}$（使用较小的 $\lambda$）

2. **步骤2**：计算自适应权重 $w_j = 1/(|\hat{\beta}_{\text{coco},j}| + 1e-8)^\gamma$

3. **步骤3**：加权惩罚项的重参数化：
   - 令 $\beta = W_{\text{diag}} \cdot \alpha$，其中 $W_{\text{diag}} = \text{diag}(w_1, w_2, ..., w_p)$
   - 代入目标函数得到变换后的优化问题：
     $$\hat{\alpha} = \arg\min_{\alpha} \frac{1}{2}\alpha^T (W_{\text{diag}}^T \tilde{\Sigma} W_{\text{diag}}) \alpha - (W_{\text{diag}}^T \tilde{\rho})^T \alpha + \lambda\|\alpha\|_1$$

4. **步骤4**：沿用CoCoLasso的ADMM投影+Cholesky分解+标准Lasso求解器

5. **步骤5**：逆变换得到原始系数估计 $\hat{\beta}_{\text{ada\_coco}} = W_{\text{diag}} \cdot \hat{\alpha}$

#### 代码文件

`src/Adaptive_CoCoLasso.py`

### 6. 随机森林修正Lasso（树方法创新算法1）

#### 模型原理

本项目提出的创新方法，使用随机森林的特征重要性作为自适应权重，替换传统的基于系数估计的权重。随机森林能够捕捉非线性特征关系，提供更稳健的特征重要性度量。

#### 核心思想

1. 使用随机森林回归器拟合 $W$ 与 $y$，提取特征重要性
2. 特征越重要，对应的自适应权重越小（惩罚越小）
3. 基于特征重要性构建权重，融入修正Lasso框架

#### 核心公式

1. **随机森林特征重要性提取**

   训练随机森林回归器：
   $$\text{RF}(W, y) \rightarrow \{\text{FI}_j\}_{j=1}^p$$

   其中 $\text{FI}_j$ 为第 $j$ 个特征的特征重要性（基于基尼不纯度减少或均方误差减少）。

2. **自适应权重计算**

   $$w_j = 1 / \left( \frac{\text{FI}_j}{\max(\text{FI})} + 1e-8 \right)^\gamma$$

   其中 $\gamma > 0$ 为权重指数，默认 $\gamma=1$。加入极小值 $1e-8$ 避免除零错误。

3. **随机森林修正Lasso目标函数**

   $$\hat{\beta}_{\text{RF\_corrected}} = \arg\min_{\beta} \frac{1}{2n}\|y - W\beta\|_2^2 - \frac{1}{2}\beta^T\Sigma_{uu}\beta + \lambda\sum_{j=1}^p w_j|\beta_j|$$

   该目标函数与自适应修正Lasso形式一致，区别仅在于权重 $w_j$ 的来源——基于随机森林特征重要性而非初始系数估计。

#### 算法实现步骤

1. **数据标准化**：对 $W$ 和 $y$ 做标准化

2. **随机森林特征重要性提取**：
   - 训练随机森林回归器（`RandomForestRegressor`）
   - 获取特征重要性 `feature_importances_`

3. **权重计算**：基于特征重要性计算自适应权重

4. **加权优化重参数化**：
   - 令 $\beta_j = w_j \cdot \alpha_j$，代入目标函数
   - 对协变量矩阵的每一列做变换：$W_j = W_j / w_j$
   - 对 $\Sigma_{uu}$ 做变换：$\Sigma_{uu}^{\text{weighted}} = \text{diag}(w) \Sigma_{uu} \text{diag}(w)$

5. **标准Lasso求解**：在变换后的数据上调用 `sklearn.linear_model.Lasso`

6. **逆变换**：$\hat{\beta}_j = w_j \cdot \hat{\alpha}_j$，变换回原始尺度

#### 代码文件

`src/RandomForest_Corrected_Lasso.py`

### 7. XGBoost修正Lasso（树方法创新算法2）

#### 模型原理

本项目提出的另一创新方法，使用XGBoost的特征重要性作为自适应权重。XGBoost作为梯度提升树模型，通常能提供更精细的特征重要性排序。

#### 核心思想

1. 使用XGBoost回归器拟合 $W$ 与 $y$，提取特征重要性
2. 特征越重要，对应的自适应权重越小（惩罚越小）
3. 基于特征重要性构建权重，融入修正Lasso框架

#### 核心公式

1. **XGBoost特征重要性提取**

   训练XGBoost回归器：
   $$\text{XGB}(W, y) \rightarrow \{\text{FI}_j\}_{j=1}^p$$

   其中 $\text{FI}_j$ 为第 $j$ 个特征的特征重要性（可基于权重、增益、覆盖等多种指标）。

2. **自适应权重计算**

   $$w_j = 1 / \left( \frac{\text{FI}_j}{\max(\text{FI})} + 1e-8 \right)^\gamma$$

   其中 $\gamma > 0$ 为权重指数，默认 $\gamma=1$。加入极小值 $1e-8$ 避免除零错误。

3. **XGBoost修正Lasso目标函数**

   $$\hat{\beta}_{\text{XGB\_corrected}} = \arg\min_{\beta} \frac{1}{2n}\|y - W\beta\|_2^2 - \frac{1}{2}\beta^T\Sigma_{uu}\beta + \lambda\sum_{j=1}^p w_j|\beta_j|$$

   该目标函数与随机森林修正Lasso形式一致，区别仅在于特征重要性由XGBoost而非随机森林提供。

#### 算法实现步骤

1. **数据标准化**：对 $W$ 和 $y$ 做标准化

2. **XGBoost特征重要性提取**：
   - 训练XGBoost回归器（`XGBRegressor`）
   - 获取特征重要性 `feature_importances_`

3. **权重计算**：基于特征重要性计算自适应权重

4. **加权优化重参数化**：
   - 令 $\beta_j = w_j \cdot \alpha_j$，代入目标函数
   - 对协变量矩阵的每一列做变换：$W_j = W_j / w_j$
   - 对 $\Sigma_{uu}$ 做变换：$\Sigma_{uu}^{\text{weighted}} = \text{diag}(w) \Sigma_{uu} \text{diag}(w)$

5. **标准Lasso求解**：在变换后的数据上调用 `sklearn.linear_model.Lasso`

6. **逆变换**：$\hat{\beta}_j = w_j \cdot \hat{\alpha}_j$，变换回原始尺度

## 实现关键注意事项

1. **数据标准化**：所有模型拟合前必须对协变量矩阵 $W$ 做标准化，拟合完成后需将系数变换回原始尺度。

2. **测量误差协方差**：加性误差场景下默认 $\Sigma_{uu}$ 已知；若 $\Sigma_{uu}$ 未知，可通过重复测量或矩估计方法获取。

3. **半正定修正**：所有涉及Cholesky分解的算法，均需确保矩阵半正定，必要时添加对角矩阵进行修正。

4. **收敛性控制**：所有迭代优化算法，均设置合理的收敛阈值和最大迭代次数，保证数值稳定性。

5. **结果可复现**：所有模拟函数默认设置固定随机种子，确保结果可复现。
