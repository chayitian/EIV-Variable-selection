# 变量选择工具包

## 项目概述

本工具包实现了三种经典的变量选择方法：**普通最小二乘（OLS）**、**Lasso回归**和**自适应Lasso回归**。这些方法构成了高维变量选择的基础框架，适用于无测量误差的标准线性回归场景。

标准线性回归模型定义：

$$y = X\beta^* + \varepsilon$$

其中：

- $y \in \mathbb{R}^n$：响应变量向量
- $X \in \mathbb{R}^{n \times p}$：协变量矩阵
- $\beta^* \in \mathbb{R}^p$：真实系数向量
- $\varepsilon \in \mathbb{R}^n$：独立零均值噪声

---

## 算法演进逻辑

```
OLS（无稀疏性）
    ↓
Lasso（L1稀疏性，无神谕性质）
    ↓
自适应Lasso（L1稀疏性 + 神谕性质）
```

---

## 核心模型与实现规范

所有模型均采用标准化的输入输出接口设计：

- **输入**：`X`（协变量矩阵）、`y`（响应变量向量）
- **输出**：`coef_`（系数估计向量）、`intercept_`（截距项估计值）

### 1. 普通最小二乘回归（OLS）

#### 模型原理

普通最小二乘法通过最小化残差平方和来估计回归系数，是最基础的回归方法。OLS不产生稀疏解，所有变量的系数均非零，因此不具备变量选择功能，但可作为其他方法的基准或初始估计。

#### 目标函数

$$\hat{\beta}_{\text{OLS}} = \arg\min_{\beta} \frac{1}{2n}\|y - X\beta\|_2^2$$

解析解：

$$\hat{\beta}_{\text{OLS}} = (X^TX)^{-1}X^Ty$$

#### 算法实现步骤

**步骤1：数据中心化**

$$\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i, \quad \bar{y} = \frac{1}{n}\sum_{i=1}^n y_i$$

$$\tilde{X} = X - \bar{X}, \quad \tilde{y} = y - \bar{y}$$

**步骤2：构建协方差矩阵**

$$\Sigma_X = \frac{1}{n}\tilde{X}^T\tilde{X}$$

**步骤3：计算协方差向量**

$$\rho = \frac{1}{n}\tilde{X}^T\tilde{y}$$

**步骤4：求解线性方程组**

$$\hat{\beta} = \Sigma_X^{-1}\rho$$

**步骤5：恢复截距项**

$$\hat{\beta}_0 = \bar{y} - \bar{X}^T\hat{\beta}$$

#### 代码文件

`src/OLS.py`

#### 使用示例

```python
from src.OLS import OLS

model = OLS(fit_intercept=True, normalize=False)
model.fit(X, y)
y_pred = model.predict(X_test)
```

### 2. Lasso回归

#### 模型原理

Lasso（Least Absolute Shrinkage and Selection Operator）通过在OLS目标函数中添加L1惩罚项，实现系数的稀疏性估计。L1惩罚能够将部分系数精确压缩为零，从而实现自动变量选择。本实现采用**坐标下降法**求解。

#### 目标函数

$$\hat{\beta}_{\text{Lasso}} = \arg\min_{\beta} \frac{1}{2n}\|y - X\beta\|_2^2 + \lambda\|\beta\|_1$$

其中：

- $\lambda > 0$：正则化参数，控制稀疏程度
- $\|\beta\|_1 = \sum_{j=1}^p |\beta_j|$：L1范数

#### 算法实现步骤

**步骤1：数据中心化**

$$\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i, \quad \bar{y} = \frac{1}{n}\sum_{i=1}^n y_i$$

$$\tilde{X} = X - \bar{X}, \quad \tilde{y} = y - \bar{y}$$

**步骤2：初始化**

初始化系数向量：$\beta^{(0)} = \mathbf{0} \in \mathbb{R}^p$

**步骤3：坐标下降法迭代**

对每个坐标 $j = 1, 2, \ldots, p$ 依次更新：

计算部分残差：
$$r_j^{(t)} = \tilde{y} - \sum_{k \neq j} \tilde{X}_k \beta_k^{(t)}$$

计算协方差：
$$\rho_j^{(t)} = \frac{1}{n}\tilde{X}_j^T r_j^{(t)}$$

计算归一化因子：
$$z_j = \frac{1}{n}\|\tilde{X}_j\|_2^2$$

软阈值更新：
$$\beta_j^{(t+1)} = \frac{S(\rho_j^{(t)}, \lambda)}{z_j}$$

其中软阈值函数：
$$S(\rho, \lambda) = \text{sign}(\rho) \cdot \max(|\rho| - \lambda, 0) = 
\begin{cases}
\rho - \lambda, & \rho > \lambda \\
0, & |\rho| \leq \lambda \\
\rho + \lambda, & \rho < -\lambda
\end{cases}$$

**步骤4：收敛判断**

迭代至满足收敛条件：
$$\|\beta^{(t+1)} - \beta^{(t)}\|_\infty < \epsilon$$

或达到最大迭代次数。

**步骤5：恢复截距项**

$$\hat{\beta}_0 = \bar{y} - \bar{X}^T\hat{\beta}$$

#### 代码文件

`src/Lasso.py`

#### 使用示例

```python
from src.Lasso import LassoRegression

model = LassoRegression(alpha=0.1, max_iter=1000, tol=1e-4)
model.fit(X, y)
y_pred = model.predict(X_test)
```

### 3. 自适应Lasso回归（Adaptive Lasso）

#### 模型原理

自适应Lasso由Zou (2006)提出，是对标准Lasso的重要改进。通过对不同系数施加差异化权重，自适应Lasso能够实现**神谕性质（Oracle Property）**：即变量选择相合性和非零系数估计的渐近正态性。

核心思想：
- 对真实值较大的系数施加较小的惩罚（权重小）
- 对真实值为零的系数施加较大的惩罚（权重大）

#### 目标函数

$$\hat{\beta}_{\text{AdaLasso}} = \arg\min_{\beta} \frac{1}{2n}\|y - X\beta\|_2^2 + \lambda\sum_{j=1}^p w_j|\beta_j|$$

其中自适应权重：
$$w_j = \frac{1}{|\hat{\beta}_j^{\text{init}}|^\gamma}, \quad \gamma > 0$$

$\hat{\beta}^{\text{init}}$ 为初始相合估计（通常使用OLS或Ridge估计）。

#### 神谕性质

当 $\gamma > 0$ 且 $\lambda_n / \sqrt{n} \to 0$，$\lambda_n n^{(\gamma-1)/2} \to \infty$ 时，自适应Lasso满足：

1. **变量选择相合性**：$\Pr\left(\{j: \hat{\beta}_j \neq 0\} = \{j: \beta_j^* \neq 0\}\right) \to 1$

2. **渐近正态性**：$\sqrt{n}(\hat{\beta}_{\mathcal{S}} - \beta_{\mathcal{S}}^*) \xrightarrow{d} N(0, \Sigma_{\mathcal{S}}^{-1})$

其中 $\mathcal{S} = \{j: \beta_j^* \neq 0\}$ 为真实非零系数的下标集合。

#### 算法实现步骤

**步骤1：数据中心化**

$$\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i, \quad \bar{y} = \frac{1}{n}\sum_{i=1}^n y_i$$

$$\tilde{X} = X - \bar{X}, \quad \tilde{y} = y - \bar{y}$$

**步骤2：初始估计**

使用OLS获取初始相合估计：
$$\Sigma_X = \frac{1}{n}\tilde{X}^T\tilde{X}$$

$$\rho = \frac{1}{n}\tilde{X}^T\tilde{y}$$

$$\hat{\beta}^{\text{init}} = \Sigma_X^{-1}\rho$$

**步骤3：计算自适应权重**

$$w_j = \frac{1}{(|\hat{\beta}_j^{\text{init}}| + \epsilon)^\gamma}$$

其中 $\epsilon > 0$ 为避免除零的小常数。

**步骤4：加权坐标下降法**

令 $\beta_j = \alpha_j / w_j$，将问题转化为标准Lasso形式：

加权协变量：
$$\tilde{X}_j^{\text{weighted}} = \tilde{X}_j \cdot w_j$$

使用坐标下降法求解：
$$\hat{\alpha} = \arg\min_{\alpha} \frac{1}{2n}\|\tilde{y} - \tilde{X}^{\text{weighted}}\alpha\|_2^2 + \lambda\|\alpha\|_1$$

**步骤5：逆变换得到最终估计**

$$\hat{\beta}_j = \frac{\hat{\alpha}_j}{w_j}$$

**步骤6：恢复截距项**

$$\hat{\beta}_0 = \bar{y} - \bar{X}^T\hat{\beta}$$

#### 代码文件

`src/Adaptive_Lasso.py`

#### 参考文献

Zou, H. (2006). The Adaptive Lasso and Its Oracle Properties. *Journal of the American Statistical Association*, 101(476), 1418-1429.

#### 使用示例

```python
from src.Adaptive_Lasso import AdaptiveLasso

model = AdaptiveLasso(alpha=0.1, gamma=1.0, max_iter=1000, tol=1e-4)
model.fit(X, y)
y_pred = model.predict(X_test)

print(f"自适应权重: {model.weights_}")
print(f"初始估计: {model.init_coef_}")
```

## 方法对比

| 方法 | 稀疏性 | 变量选择 | 神谕性质 | 适用场景 |
|------|--------|----------|----------|----------|
| OLS | 否 | 否 | - | 低维回归、基准模型 |
| Lasso | 是 | 是 | 否 | 高维变量选择 |
| 自适应Lasso | 是 | 是 | 是 | 高维变量选择（理论最优） |

## 统一接口

所有模型遵循sklearn风格的API设计：

```python
class ModelName:
    def __init__(self, ...):
        ...
    
    def fit(self, X, y):
        """
        拟合模型
        
        参数
        ----------
        X : np.ndarray, shape (n_samples, n_features)
            协变量矩阵
        y : np.ndarray, shape (n_samples,)
            响应变量向量
        
        返回
        -------
        self : object
            拟合后的模型实例
        """
        ...
    
    def predict(self, X):
        """
        预测
        
        参数
        ----------
        X : np.ndarray, shape (n_samples, n_features)
            协变量矩阵
        
        返回
        -------
        y_pred : np.ndarray, shape (n_samples,)
            预测值
        """
        return X @ self.coef_ + self.intercept_
```

## 数学符号说明

| 符号 | 含义 |
|------|------|
| $n$ | 样本数量 |
| $p$ | 特征数量 |
| $X$ | 协变量矩阵 $(n \times p)$ |
| $y$ | 响应变量向量 $(n \times 1)$ |
| $\beta$ | 系数向量 $(p \times 1)$ |
| $\beta_0$ | 截距项 |
| $\lambda$ | 正则化参数 |
| $\gamma$ | 自适应权重指数 |
| $w_j$ | 第 $j$ 个特征的自适应权重 |
| $\|\cdot\|_2$ | L2范数 |
| $\|\cdot\|_1$ | L1范数 |
| $\Sigma_X$ | 协变量的样本协方差矩阵 |
| $\rho$ | 协变量与响应变量的协方差向量 |
| $S(\cdot, \cdot)$ | 软阈值函数 |
